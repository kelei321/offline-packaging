# uts-kux-mlkit-scancode

来源：`unpackage/resources/app-android/uni_modules/kux-mlkit-scancode`

同步脚本会复制：

- `src/` -> `src/main/java/`
- `res/` -> `src/main/res/`
- `assets/` -> `src/main/assets/`
- `libs/` -> `libs/`
- `AndroidManifest.xml` -> `src/main/AndroidManifest.xml`，并移除 manifest package

该插件依赖 CameraX、ML Kit，并包含自带 AAR。
